# Interactionary

An Epic Fight addon featuring zero-bend animations and full weapon reworks that shouldn't be in Epic Fight. Built for a custom modpack. Free to use in your own modpacks, but please give credit.

## !! Disclaimer
Most of the animations featured in this addon are made by me, but the others were made by PierceTH by request of me. The animations are already featured in [Epic Fight Plus - Pierced NoBends](https://www.curseforge.com/minecraft/texture-packs/epic-fight-plus-pierced-nobends). Pierce gets half my revenue from this mod.
